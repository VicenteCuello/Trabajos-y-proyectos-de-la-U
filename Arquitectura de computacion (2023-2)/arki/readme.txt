Para ejecutar este programa escrito en C, sigue estos pasos generales:

Instala un compilador C: Necesitas un compilador C para convertir el código fuente C en un programa ejecutable. En sistemas Unix/Linux, el compilador C más común es GCC (GNU Compiler Collection). En Windows, puedes usar MinGW (Minimalist GNU for Windows) o instalar un IDE como Code::Blocks, Dev-C++, o Visual Studio con soporte C/C++.(se recomienda DEV-c++ por su simpleza)



Compila el código fuente: Abre una terminal o línea de comandos y navega hasta la ubicación del archivo .c. Luego, utiliza el compilador para compilar el código. Por ejemplo, si estás utilizando GCC en una terminal Unix/Linux, puedes hacerlo de la siguiente manera:


gcc -o mi_programa mi_programa.c

Esto compilará el archivo "mi_programa.c" y generará un ejecutable llamado "mi_programa" (o el nombre que elijas después de -o).

Ejecuta el programa: Una vez compilado con éxito, puedes ejecutar el programa escribiendo su nombre en la terminal y presionando Enter. Por ejemplo: ./mi_programa

En Windows, simplemente escribe el nombre del programa sin el prefijo "./" y presiona Enter:


Estos son los pasos básicos para compilar y ejecutar un programa escrito en C. Asegúrate de que tengas el compilador C adecuado instalado en tu sistema y de que estés trabajando en el directorio correcto donde se encuentra tu archivo fuente antes de compilarlo. Además, ten en cuenta que algunos IDEs proporcionan funciones de compilación y ejecución integradas para simplificar este proceso.

Cabe destacar que el codigo no utiliza librerias especiales ni instrucciones muy complejas por lo que no debiera dar mayores problemas para su ejecucion.
